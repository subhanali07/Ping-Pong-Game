# 🏓 Arcade Ping Pong Game

A classic, retro-inspired 2D Ping Pong game built entirely in Python. This project features a local two-player experience with real-time keyboard controls, dynamic collision physics, and an automated scoreboard system.

## 🚀 Features

- **Local Multiplayer:** Play simultaneously with a friend on a single keyboard.
- **Dynamic Physics:** Smooth ball movement with axis-aligned collision detection against paddles and wall boundaries.
- **Real-Time Score Tracking:** Integrated canvas scoreboard that automatically updates whenever a player scores.
- **Responsive Controls:** Quick and snappy paddle movement handling.

---

## 🎮 Controls

The game is designed for two players sitting at the same keyboard:

| Action | Player 1 (Left Paddle) | Player 2 (Right Paddle) |
| :--- | :--- | :--- |
| **Move Up** | `W` | `Up Arrow` |
| **Move Down**| `S` | `Down Arrow` |
